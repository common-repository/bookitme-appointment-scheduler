BOOKITME APPOINTMENT SCHEDULER plugin is easy interface to include shortcode in any PAGE or POST for display appointment scheduler from bookitme.com

== Description ==

The **Bookitme** lets you add online time sheet booking or appointments and reservation handling to your website
It is simple way to display appointments scheduler in post or page. 
Simply adding a shortcode **[bookitme_appointments]** to a page. 
See the Plugin screenshots for examples.

**FREE LIMITED EDITON**

**Sent email into sales@bookitme.com for your API request**

**NEW BOOKITME - APPOINTMENT SCHEDULER API WILL BE GENERATED DEMAND ON YOUR REQUEST from email sales@bookitme.com**

This gives your visitors an efficient way to view ALL appointment scheduler from bookitme.com site on ONE place. 
Shortodes:
**[bookitme_appointments]** – display calendar on page or post 
params: [bookitme_appointments width="500" height="500" url="http://bookitme.com/bookitme-appointment/appointment.html"]

Admin Page: all shortcode input parameters are setup in admin page. If you not specified input parameters in shortcode definition default values from Admin panel will be used.

== Installation ==
Instructions for installing Bookitme Appointment Scheduler Plugin.
1. Download and extract the Plugin zip file.
2. Upload the folder containing the Plugin files to your WordPress Plugins folder (usually ../wp-content/plugins/ folder).
3. Activate the Plugin via the 'Plugins' menu in WordPress.
4. Add [bookitme_appointments] shortcode to a page to display booking calendar on your site.
The Plugin can also be installed directly from the main WordPress Plugin page.

1. Go to the Plugins => Add New page.
2. Enter 'Bookitme'  in the textbox and click the 'Search Plugins' button.
3. In the list of relevant Plugins click the 'Install' link for Simple Sitemap on the right hand side of the page.
4. Click the 'Install Now' button on the popup page.
5. Click 'Activate Plugin' to finish installation.
6. Add [bookitme_appointments] shortcode to a page to display the sitemap on your site.
